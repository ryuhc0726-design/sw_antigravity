# 독서 기록 관리 프로그램
이 프로그램은 독서를 좋아하는, 독서를 시작하고자 하는 모든 사람들을 위한 프로그램입니다.

## 주요 기능
- 읽은 책, 책의 저자, 읽은 날짜, 읽은 쪽수를 기록할 수 있습니다.
- 책을 읽으며 떠올린 생각, 인상깊은 구절 등을 감상문에 기록할 수 있습니다.
- 그래프를 통해 자신의 독서 기록을 주별, 월별로 한눈에 파악할 수 있습니다.

### 독서 기록 예시
|책 제목|저자|읽은 날짜|쪽수|
|--|--|--|--|
|죄와 벌|도스토옙스키|6월 13일|50쪽|
